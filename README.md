## Quick Tips

- Think of Claude as an AGI partner, not just a tool
- Always work in plan mode for complex tasks
- Create slash commands for repetitive workflows
- Use the Ralph Loop for autonomous, long-running tasks
- Verify work regularly with diligence prompts
- Create a "CEO" prompt for each major project
