## How to Use This Template

1. Copy this file to your project directory
2. Fill in the task, requirements, and instructions
3. Customize completion criteria for your specific needs
4. Run with: `/ralph-loop --prompt PROMPT.md --max-iterations 15 --completion-promise "COMPLETE"`
