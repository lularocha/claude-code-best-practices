# Glossary

**AGI - Artificial General Intelligence**<br>
AI systems that can understand and perform any intellectual task a human can do

**AI - Artificial Intelligence**<br>
Computer systems designed to perform tasks that typically require human intelligence

**API - Application Programming Interface**<br>
A set of protocols for building and integrating software

**CI - Continuous Integration**<br>
Practice of automatically building and testing code changes

**CLI - Command Line Interface**<br>
A text-based interface for interacting with software

**CSS - Cascading Style Sheets**<br>
Language for describing the presentation of web pages

**MCP - Model Context Protocol**<br>
A protocol that connects Claude to external tools and data sources

**PR - Pull Request**<br>
A proposal to merge code changes into a repository

**PRD - Product Requirements Document**<br>
A document describing features and functionality of a product

**UI - User Interface**<br>
The visual elements through which users interact with software
